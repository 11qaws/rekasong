## 공통 우선순위

1. **P0 접근성·명료성:** 토스트를 보조기기에 알리고, 로고의 문자형 장식은 CSS 도형으로 교체한다.
2. **P1 운영 신뢰:** 동기화 연결 상태, 대기열 삭제 Undo, 긴급 정지의 영향 설명을 다음 감사의 핵심 후보로 둔다.
3. **P1 첫 실행 정책:** 유튜브 검색을 새 사용자 기본 탭으로 유지하면서 재방문자의 마지막 탭 복원 규칙을 명문화한다.
4. **P2 시각 QA:** 모바일·긴 텍스트·저사양 애니메이션을 실제 viewport로 검증한다.
